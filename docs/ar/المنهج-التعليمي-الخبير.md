# المنهج التعليمي الموجّه + خبرات مستوردة

## ما الجديد؟

| المكوّن | الوظيفة |
|---------|---------|
| `expert-curriculum.json` | 4 مراحل موجهة (نظام → micro → دمج → إتقان) |
| `expert-weights-preset.json` | أوزان خبراء (Murphy/Chan/microstructure) |
| **Expert consensus** | لا up/down إلا بـ 2+ استراتيجيات S-tier |
| `agent:curriculum` | سلسلة تعليم كاملة مع اختبار hit rate كل مرحلة |

## شغّل المنهج (جهازك)

```powershell
cd C:\Users\pc\zambahola-one-agent
git pull origin main
npm run agent:curriculum
```

**وقت طويل** — 4 مراحل + آلاف الشموع. النتيجة في:
`apps/one-agent/data/learning/curriculum-progress.json`

## بعد المنهج

```powershell
npm run agent:stop
npm run agent:max-accuracy:start
```

تأكد في اللوحة: `expertMode: true` في التنبؤ.

## لماذا كان Hit Rate منخفض؟

- تعلم عشوائي بدون **طبقات خبراء**
- استراتيجيات متضاربة (mean-rev ضد ترند)
- المنهج يمنع ذلك ويرفع أوزان S-tier

## المراحل الأربع

1. **P1_regime** — ترند/رنج/تقلب (3000 شمعة)
2. **P2_micro** — order book + funding + OI (5000)
3. **P3_fusion** — دمج Chan/Prado (8000)
4. **P4_mastery** — معايرة نهائية (10000)

كل مرحلة لها **حد أدنى hit rate** — إن فشلت تُسجَّل في progress.
