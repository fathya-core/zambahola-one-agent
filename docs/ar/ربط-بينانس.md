## المرحلة 1 — تعليم زيادة (الآن)

**الوكيل شغّال؟** افتح نافذة PowerShell ثانية:

```powershell
cd C:\Users\pc\zambahola-one-agent
git pull origin main
npm run agent:teach-intensive
```

هذا يشغّل: teach-more (20) + deep (10) + mega (5000) + ultra (12) + export.

أو يدوي:

```powershell
$env:ZAMBAHOLA_TEACH_CYCLES="25"
npm run agent:teach-more
```

## المرحلة 3 — Binance Demo (Testnet)

1. أنشئ مفاتيح من: https://testnet.binancefuture.com
2. انسخ `config/exchange-demo.env.example` → `exchange.env` (لا ترفعه GitHub)
3. عبّي `BINANCE_API_KEY` و `BINANCE_API_SECRET`

```powershell
npm run agent:stop
# حمّل المتغيرات (مثال PowerShell):
Get-Content exchange.env | ForEach-Object {
  if ($_ -match '^([^#=]+)=(.*)$') { Set-Item -Path "env:$($matches[1])" -Value $matches[2] }
}
$env:ZAMBAHOLA_BROKER="binance_demo"
npm run agent:max-accuracy:start
```

الوكيل يرسل أوامر **testnet** + يتابع PnL ورقي.

## المرحلة 4 — حساب حقيقي (بعد استقرار Demo)

**شروط قبل التفعيل:**
- Hit rate مستقر على demo ≥ 2 أسبوع
- `ZAMBAHOLA_STABILIZE=1` شغّال
- حجم صغير: `ZAMBAHOLA_ORDER_QTY=0.001`

```powershell
$env:ZAMBAHOLA_BROKER="binance_live"
$env:ZAMBAHOLA_I_ACCEPT_REAL_TRADING="RISK"
$env:ZAMBAHOLA_BINANCE_TESTNET="0"
# مفاتيح LIVE في exchange.env فقط
npm run agent:max-accuracy:start
```
