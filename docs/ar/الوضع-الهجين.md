# الوضع الهجين — `phase2-hybrid`

يبدّل الوكيل **تلقائياً** بين التعلّم والدقة حسب السوق.

## كيف يعمل؟

| نظام السوق (regime) | الوضع | ماذا يفعل |
|---------------------|-------|-----------|
| **range** (عرضي) | **learn** | paper trades + lean + أفق 25s |
| **trend_up / trend_down / high_vol** | **signals** | بوابات دقة + meta + أفق 45s |

التبديل بعد **12 tick** متتالية من نفس النظام (حتى ما يتذبذب كل ثانية).

---

## التشغيل

```powershell
cd C:\Users\pc\zambahola-one-agent
git pull origin main
npm run agent:stop
npm run agent:phase2-hybrid
```

## ليلاً (8 ساعات)

```powershell
npm run agent:overnight-learn
```

الآن الـ overnight يشغّل **hybrid** افتراضياً.

---

## ماذا تراقب في اللوحة؟

| الحقل | المعنى |
|-------|--------|
| **Hybrid auto** | ON |
| **Hybrid profile** | `learn (range)` أو `signals (trend)` |
| **Paper trades** | يزيد في وضع learn |
| **Directional hit** | يُقاس في وضع signals |
| **Horizon** | 25s في learn · 45s في signals |

المحلل العربي يكتب: `هجين: تعلّم + تداول ورقي` أو `إشارات دقيقة`.

---

## متى «خلص التعلّم»؟

1. **Paper trades > 100** في جلسات learn
2. **Hybrid profile** يتحول لـ `signals` لما السوق ترند
3. **Directional hit ≥ 58%** في فترات signals
