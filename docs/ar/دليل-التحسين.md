## التشغيل

```bash
npm run setup
npm run agent:start          # Binance افتراضي
ZAMBAHOLA_FEED=mock npm run agent:start   # محاكاة
npm run agent:learn          # 5 دورات تعلم
npm run agent:turbo-learn    # تعلم سريع mock
npm run agent:stop
```

## متغيرات البيئة

| المتغير | القيم | المعنى |
|---------|-------|--------|
| `ZAMBAHOLA_FEED` | `binance` / `mock` | مصدر السعر |
| `ZAMBAHOLA_LEARN_CYCLES` | رقم | دورات `agent:learn` |

## واجهة API

- `/api/sentiment` — المشاعر الحالية
- `/api/strategies` — أوزان + إحصائيات
- `/api/knowledge` — الفهرس الكامل

npm run agent:stop
