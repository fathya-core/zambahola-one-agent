# دليل تحسين دقة التنبؤ — ZAMBAHOLA

## ما تم إضافته

1. **ست استراتيجيات** مجمّعة: زخم، عودة للمتوسط، RSI، تقاطع EMA، نظام التقلب، بولينجر.
2. **تصويت مجمع (Ensemble)** مع درجة اتفاق بين الاستراتيجيات.
3. **أوزان تكيفية** تتعلم من نتائج `prediction_hit` لكل استراتيجية.
4. **فهرس معرفة** في `apps/one-agent/knowledge/INDEX.json`: كتب (مراجع قانونية)، مواقع، أخبار، تقارير.
5. **سجل بحث** `knowledge/research-log.jsonl` يُحدَّث تلقائياً عند كل تقييم.

## أوامر

```bash
npm run setup
npm run agent:start
npm run agent:learn      # دورات تعلم متعددة
npm run agent:test-run
```

## مكتبة المراجع

افتح الفهرس:

```bash
cat apps/one-agent/knowledge/INDEX.json
```

أو من الواجهة: `http://localhost:8787/api/knowledge`

## تنبيه

- لا تداول حقيقي في v0.
- لا ننسخ نصوص كتب محمية — مراجع وروابط فقط.
- الدقة تتحسن بالتجربة والبيانات الحية، وليس بضمان ثابت.

## التوقف

عندما تطلب التوقف، أوقف الوكيل:

```bash

```
