# أقصى نسبة تنبؤ — على جهازك

## خطوة واحدة (بعد `npm run setup`)

```bash
npm run agent:max-accuracy
```

هذا يشغّل **تدريب كامل** بأقصى إعدادات (30+25+25 دورة، 10k شموع، فلاتر دقة).

## تشغيل حي بعد التدريب

```bash
npm run agent:max-accuracy:start
```

ثم افتح: **http://127.0.0.1:8787**

## ماذا يفعل وضع `max`؟

| الإعداد | القيمة |
|---------|--------|
| التغذية | `universal` + تيكات سريعة Binance |
| أفق التنبؤ | 45 ثانية (أقل ضوضاء) |
| الفلتر | لا up/down إلا باتفاق ≥58% و3 نماذج ML |
| التدريب | 10,000 شمعة + ultra-learn |

انسخ `config/max-accuracy.env` إلى `.env` إن أردت تعديل القيم.

## صراحة تقنية

لا يوجد نظام يضمن **100%** إصابة. وضع max يشدّد **التشغيل الحي** فقط (`ZAMBAHOLA_ACCURACY_FILTER=live`): إشارات up/down عند اتفاق قوي. أثناء التدريب الفلتر مُوقف حتى تتعلم النماذج. راقب في اللوحة: `hitRate` و `qualityTier` و `directionalHitRate` في backtest.

## إن Binance محجوب عندك

```bash
ZAMBAHOLA_FEED=bybit_primary npm run agent:max-accuracy:start
```
