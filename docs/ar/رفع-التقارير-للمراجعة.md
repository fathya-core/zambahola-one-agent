# رفع التقارير للمراجعة — ثم تعليم الوكيل

## مرحلتان

| المرحلة | الأمر | النتيجة |
|---------|-------|---------|
| **1 — مراجعة AI** | `collect-downloads-reports.ps1` | ملف واحد ترفعه في Cursor |
| **2 — تعليم الوكيل** | `agent:import-md-reports` + `omni-train` | بعد موافقتك |

---

## الكود الجاهز (PowerShell — مرة واحدة)

```powershell
cd C:\Users\pc\zambahola-one-agent
git pull origin main
npm run setup
.\scripts\ops\collect-downloads-reports.ps1
```

يفتح لك ملف:
`apps\one-agent\knowledge\user-reports\BUNDLE-FOR-REVIEW.md`

### ارفعه في Cursor
- اسحب الملف إلى المحادثة، أو
- `@BUNDLE-FOR-REVIEW.md` في الشات

اكتب: **«راجع الحزمة وقلي شو نعطي الوكيل»**

---

## بدون سكربت PowerShell (يدوي)

```powershell
mkdir apps\one-agent\knowledge\user-reports -Force
copy C:\Users\pc\Downloads\ايه.md apps\one-agent\knowledge\user-reports\
npm run agent:bundle-reports -- C:\Users\pc\Downloads
```

أو من المجلد بعد النسخ:

```powershell
npm run agent:bundle-reports
```

---

## بعد موافقة AI

```powershell
npm run agent:import-md-reports
npm run agent:omni-train
npm run agent:max-accuracy:start
```
