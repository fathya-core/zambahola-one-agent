# التشغيل الليلي (8 ساعات نوم)

## أمر واحد قبل النوم

```powershell
cd C:\Users\pc\zambahola-one-agent
git pull origin main
npm run agent:overnight-learn
```

**صغّر نافذة PowerShell — لا تقفلها.** إذا قفلتها قد يتوقف المراقب (الوكيل نفسه يبقى detached غالباً، لكن الرفع التلقائي يتوقف).

---

## ماذا يحدث أثناء نومك؟

| كل | الإجراء |
|----|---------|
| **~90 ثانية** | يتأكد الوكيل شغّال — إذا وقف يعيد تشغيله |
| **~45 دقيقة** | يرفع telemetry لـ GitHub تلقائياً |
| **مستمر** | `phase2-learn-trade` — paper trades + تعلّم |

---

## قبل النوم (مهم)

1. **وصّل اللابتوب بالكهرب**
2. **إعدادات Windows → الطاقة → عدم السكون** (أو السكربت يعطّل السكون على AC)
3. **لا تغلق نافذة PowerShell** — صغّرها فقط
4. **إنترنت شغّال**

```powershell
# حالة سريعة
npm run agent:health-check
curl http://127.0.0.1:8787/api/metrics

# ارفع آخر لقطة
npm run agent:push-telemetry:ps1
```

افتح اللوحة: `http://127.0.0.1:8787`  
راقب: **Paper trades** · **Learn-trade ON** · **Paper PnL**
