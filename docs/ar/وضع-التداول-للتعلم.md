# وضع التداول للتعلّم — `phase2-learn-trade`

عندما **ما عندك وقت** تنتظر ساعات والوكيل يمتنع (abstain 100%)، شغّل هذا الوضع:

## الفرق

| | phase2-signals | **phase2-learn-trade** |
|--|----------------|------------------------|
| الهدف | دقة عالية | **تعلّم سريع** |
| paper trades | نادر | **كثير** |
| أفق التقييم | 45s | **25s** |
| إغلاق الصفقة | إشارة عكسية | + **تلقائي بعد 35s** |
| range | امتناع | **lean → up/down** |

## التشغيل (OMAR-PC)

```powershell
cd C:\Users\pc\zambahola-one-agent
git pull origin main
npm run agent:stop
npm run agent:phase2-learn-trade
```

نافذة ثانية (اختياري):

```powershell
npm run agent:local-bridge
npm run agent:push-telemetry:ps1
```

## ماذا تراقب؟

| المقياس | الهدف |
|---------|--------|
| **Learn-trade** | ON |
| **Paper trades** | يزيد كل دقائق |
| **Paper PnL** | يتحرك (مو شرط ربح — المهم التعلّم) |
| **closedTradeCount** | > 10 بعد ~30 دقيقة |

## بعد التعلّم

ارجع لوضع الإشارات أو الحي:

```powershell
npm run agent:stop
npm run agent:phase2-signals
```

**ملاحظة:** هذا وضع **ورقي** فقط — لا تداول حقيقي على Binance.
