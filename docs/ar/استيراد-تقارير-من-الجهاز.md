# استيراد تقارير Perplexity من مجلد Downloads

لا يمكن للوكيل السحابي قراءة `C:\Users\pc\Downloads` مباشرة — **انسخ الملفات إلى المشروع**.

## خطوات سريعة (PowerShell)

```powershell
cd C:\Users\pc\zambahola-one-agent
git pull origin main
npm run setup

mkdir apps\one-agent\knowledge\user-reports -Force

copy C:\Users\pc\Downloads\تنبوء-2.md apps\one-agent\knowledge\user-reports\
copy "C:\Users\pc\Downloads\ابي تتعمق اكثر وتجيب اكثر عن دقة التنبوء ورفعه.md" apps\one-agent\knowledge\user-reports\دقة-التنبوء.md
copy "C:\Users\pc\Downloads\ابي كل ما يخص تعلم الاله و التداول بالذكاء الاصطنا.md" apps\one-agent\knowledge\user-reports\تعلم-الالة.md
copy C:\Users\pc\Downloads\ايه.md apps\one-agent\knowledge\user-reports\
copy "C:\Users\pc\Downloads\ايه بس بدون تدقيق لل قيود لانه انا الي ابكتبها بمل.md" apps\one-agent\knowledge\user-reports\قيود-يدوية.md

npm run agent:import-md-reports
npm run agent:omni-train
npm run agent:max-accuracy:start
```

## ماذا نستفيد من كل تقرير؟

| التقرير | المتوقع استخراجه |
|---------|------------------|
| تنبوء-2 | أوزان استراتيجيات، أفق التنبؤ |
| دقة التنبوء ورفعه | meta-labeling، abstention، hit targets |
| تعلم الآلة والتداول | أوزان ensemble، walk-forward |
| ايه / قيود يدوية | قواعدك الخاصة — تُسجَّل كـ manual |

## إذا لم يستخرج شيء

تأكد أن التقرير فيه **JSON** داخل:

````
```json
{ "weightAdjustments": { "momentum": 1.2 } }
```
````

أو أضف رد Perplexity منفصل:

```powershell
npm run agent:research-import -- perplexity-answer.json
```

## بعد الاستيراد

راقب في اللوحة: **Directional hit** و **Dir. rolling (60)**.
