# المجلد موجود لكن الأوامر لا تعمل؟

معناها النسخة **قديمة** (قبل دمج المشروع الكامل).

## PowerShell — داخل المجلد `zambahola-one-agent`

```powershell
cd C:\Users\pc\zambahola-one-agent
git fetch origin
git checkout main
git pull origin main
npm run setup
npm run agent:max-accuracy
```

بعد انتهاء التدريب:

```powershell
npm run agent:max-accuracy:start
```

ثم المتصفح: http://127.0.0.1:8787

## إذا `git pull` فيه تعارض

```powershell
git stash
git pull origin main
git stash pop
```

## إذا ما فيه `git` أو المجلد ليس من GitHub

احذف المجلد القديم واستنسخ من جديد:

```powershell
cd C:\Users\pc
Remove-Item -Recurse -Force zambahola-one-agent
git clone https://github.com/fathya-core/zambahola-one-agent.git
cd zambahola-one-agent
npm run setup
```
