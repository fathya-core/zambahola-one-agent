# تثبيت إضافات Cursor Marketplace لـ ZAMBAHOLA

السحابة **ما تقدر تضغط "Browse Marketplace" على جهازك**. التثبيت من **Cursor Desktop** عندك.

---

## الطريقة 1 — من الشات (الأسرع)

افتح شات Cursor في مجلد المشروع والصق **سطراً سطراً**:

```
/add-plugin tavily
/add-plugin zapier
/add-plugin huggingface-skills
/add-plugin cli-for-agent
/add-plugin cursor-sdk
/add-plugin agent-compatibility
/add-plugin continual-learning
```

بعد كل سطر اضغط Enter وانتظر حتى يثبت.

---

## الطريقة 2 — من الواجهة

1. **Settings → Plugins** (أو الزر **Browse Marketplace** في "No skills available")
2. ابحث وثبّت:

| الإضافة | لماذا نحتاجها |
|---------|----------------|
| **tavily** | بحث سوق، أخبار، استخراج docs (بديل Perplexity) |
| **zapier** | Slack · Sheets · GitHub تنبيهات |
| **huggingface-skills** | أوراق DeepLOB/TLOB · datasets · تدريب |
| **cli-for-agent** | أوامر `npm run agent:*` للوكيل |
| **cursor-sdk** | تشغيل Cloud Agents برمجياً |
| **agent-compatibility** | فحص جاهزية المشروع للوكلاء |
| **continual-learning** | تحديث AGENTS.md تلقائياً |

---

## بعد التثبيت — MCP

```powershell
cd C:\Users\pc\zambahola-one-agent
.\scripts\install-local-stack.ps1
```

أو يدوياً:

```powershell
Copy-Item .cursor\mcp.json.example .cursor\mcp.json
```

ثم **Settings → Tools & MCP → Connect** لكل سيرفر:
- ✅ `zambahola-local` (أخضر بعد تشغيل bridge)
- ✅ `zapier` — سجّل دخول Zapier
- ✅ `huggingface-skills` — سجّل HF
- ✅ `tavily` — قد يطلب `tvly login` في الطرفية

---

## تفعيل أدوات Zapier المطلوبة

في الشات اكتب:

```
فعّل Zapier: GitHub, Google Sheets, Slack, Gmail
```

أو من MCP: `discover_zapier_actions` ثم `enable_zapier_action`.

**مقترح للمشروع:**
- GitHub — issues / ملفات
- Google Sheets — سجل metrics يومي
- Slack — تنبيه عند directional ≥ 58%
