## لقطة OMAR-PC — تنبؤ منخفض (امتناع 100%)

| المقياس | قيمتك | السبب |
|---------|-------|--------|
| Hit ~32% | ضعيف | السوق **range** — استراتيجيات النطاق دقة ~42% |
| Directional 0% | خطر | الوكيل يطلع **range** فقط → ما في up/down |
| Abstain 100% | متوقع | كل تنبؤ range يُحسب **امتناع** (`range_abstain`) |
| اتفاق 53% | على الحد | كان يُمنع اتجاهي إذا block=0.62 — **تم تخفيفه** |

**مو عطل:** الوكيل **حذر** في سوق عرضي. الحل: profile أوسع إشارات + أوزان بحث + انتظار ترند.

```powershell
git pull origin main
npm run agent:stop
npm run agent:research-import -- apps/one-agent/knowledge/user-reports/AGENT-IMPORT-FINAL.json
npm run agent:phase2-signals
```
