# الوكيل الذكي — المرحلة 1

## ما أُضيف

| الميزة | الوظيفة |
|--------|---------|
| **Meta-label** | يتعلم: «هل أثق بهالإشارة؟» قبل التداول |
| **Pattern Journal** | يحلل: regime × strategy — ليه يضعف في range |
| **تحليل عربي** | يظهر في اللوحة + `pattern-journal.md` |

## Backtest — مُزال

لا يوجد `agent:backtest` ولا `mega-backtest` ولا `deep-backtest`.  
التقييم الآن **حي فقط** (اللوحة + `directional hit` + `agent:patterns`).

## التشغيل

```powershell
git pull origin main
npm run agent:stop
.\scripts\restart-directional.ps1
```

## شوف التحليل

```powershell
npm run agent:patterns
```

أو في اللوحة: قسم **التعلم المستمر** → `تحليل الأنماط`

أو الملف: `apps/one-agent/data/learning/pattern-journal.md`
