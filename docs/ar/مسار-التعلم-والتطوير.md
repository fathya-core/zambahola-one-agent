# مسار التعلم والتطوير — ZAMBAHOLA

## المراحل

1. **تشغيل حي** — وكيل ورقي + لوحة 8787 (`coingecko` أو `universal` حسب الشبكة).
2. **تعلم** — 25 دورة افتراضياً، تحديث أوزان الاستراتيجيات.
3. **تعلم عميق** — معايرة + backtest موسّع.
4. **ميجا** — تدريب على آلاف الشموع.
5. **ألترا** — خط أنابيب كامل (5000 شمعة + 30 دورة).
6. **التالي** — دقة أعلى، تغذية أسرع، تصدير نماذج (v0.7).

## استئناف سريع

من جذر المستودع: `npm run agent:path-resume` (يُشغّل الوكيل الحي ثم 5 دورات تعلم).

## ملاحظة السحابة

إذا Binance محجوب: `ZAMBAHOLA_FEED=coingecko` للسعر الحي؛ التعلم الدفعي يستخدم `mock` أو synthetic klines.
