# المهارات والروابط — دليل كامل لـ ZAMBAHOLA

هذا الملف يجمع **كل المهارات (Skills)** و**سيرفرات MCP** و**الروابط** المفيدة للمشروع — للوكيل السحابي ولـ Cursor على OMAR-PC.

---

## 1) إضافات Cursor Marketplace (ثبّتها على جهازك)

انسخ في شات Cursor سطراً سطراً:

```
/add-plugin tavily
/add-plugin zapier
/add-plugin huggingface-skills
/add-plugin cli-for-agent
/add-plugin cursor-sdk
/add-plugin agent-compatibility
/add-plugin continual-learning
```

أو شغّل: `.\scripts\install-cursor-marketplace.ps1`

| الإضافة | متى تستخدمها |
|---------|----------------|
| **tavily** | بحث ويب، أخبار، استخراج docs، بحث عميق |
| **zapier** | Slack · Sheets · GitHub · Gmail (9000+ تطبيق) |
| **huggingface-skills** | أوراق ML · نماذج · datasets · تدريب على HF Jobs |
| **cli-for-agent** | تصميم/إصلاح أوامر `npm run agent:*` |
| **cursor-sdk** | تشغيل Cloud Agents من سكربتات/CI |
| **agent-compatibility** | فحص جاهزية المستودع للوكلاء |
| **continual-learning** | تحديث `AGENTS.md` من الجلسات |

## 2) مهارات Tavily (بعد تثبيت الإضافة)

| المهارة | الاستخدام |
|---------|-----------|
| `tavily-search` | بحث سريع: «ابحث عن»، «وش آخر أخبار» |
| `tavily-research` | تقرير عميق متعدد المصادر مع اقتباسات |
| `tavily-extract` | استخراج markdown من URL محدد |
| `tavily-crawl` | تنزيل قسم كامل من موقع (docs) |
| `tavily-cli` | تثبيت/مصادقة CLI: `tvly login` |
| `tavily-best-practices` | أفضل ممارسات تكامل Tavily في الوكلاء |

**روابط:** https://tavily.com · https://docs.tavily.com · تثبيت CLI: `curl -fsSL https://cli.tavily.com/install.sh | bash`

## 3) مهارات Hugging Face

| المهارة | الاستخدام |
|---------|-----------|
| `hf-cli` | رفع/تنزيل نماذج وdatasets عبر `hf` |
| `hugging-face-datasets` | إنشاء/إدارة datasets على Hub |
| `hugging-face-dataset-viewer` | استعلام API Dataset Viewer |
| `hugging-face-model-trainer` | SFT/DPO/GRPO على HF Jobs (TRL) |
| `hugging-face-jobs` | مهام GPU عامة على HF Infrastructure |
| `hugging-face-evaluation` | نتائج eval في model cards |
| `hugging-face-paper-publisher` | نشر أوراق على Hub |
| `hugging-face-trackio` | لوحة metrics أثناء التدريب |
| `hugging-face-tool-builder` | سكربتات أتمتة ضد HF API |
| `huggingface-gradio` | واجهات Gradio للديمو |

**أدوات MCP (Huggingface-skills):**

| أداة | الغرض |
|------|--------|
| `paper_search` | بحث أوراق (DeepLOB · TLOB · order flow) |
| `hub_repo_search` | بحث models/datasets/spaces |
| `hub_repo_details` | تفاصيل repo |
| `hf_doc_search` / `hf_doc_fetch` | توثيق HF |
| `hf_hub_query` | استعلامات Hub متقدمة |
| `space_search` / `dynamic_space` | Spaces ومهام جاهزة |
| `hf_whoami` | حساب HF الحالي |

**روابط:** https://huggingface.co · https://huggingface.co/docs · MCP: `https://huggingface.co/mcp?login`

**استيراد للمشروع:**

```powershell
npm run agent:import-hf-research
npm run agent:research-import -- apps/one-agent/knowledge/hf-research-import.json
```

## 4) مهارات Zapier

| المهارة | الاستخدام |
|---------|-----------|
| `zapier-setup` | أول تشغيل، مصادقة، تفعيل تطبيقات |
| `zapier-status` | فحص صحة MCP وأدوات مفعّلة |
| `create-my-tools-profile` | skill مخصص من أدوات Zapier المفعّلة |

**أدوات MCP (بعد Connect في Cursor):**

| أداة | الغرض |
|------|--------|
| `discover_zapier_actions` | اكتشاف actions لتطبيق |
| `enable_zapier_action` / `disable_zapier_action` | تفعيل/إيقاف |
| `list_enabled_zapier_actions` | قائمة المفعّل |
| `execute_zapier_read_action` | قراءة (GitHub issues، Sheets…) |
| `execute_zapier_write_action` | كتابة (Slack، Sheets…) |
| `list_zapier_skills` / `get_zapier_skill` | workflows محفوظة |

**مقترح ZAMBAHOLA:** GitHub + Google Sheets + Slack — تنبيه عند directional ≥ 58%.

**رابط:** https://mcp.zapier.com · إعداد: Cursor → Settings → Tools & MCP → Connect Zapier
